      function addNums() {

    • var 语句 用于声明一个函数范围或全局范围的变量，并可将其初始化为一个值（可选）。
    • 在这个prompt中 返回的数据类型是字符串，+可以将字符串连接在一起
    • 如果想使用数字的话 要进行变量的转换
    • function addNums() {
      var num1 = Number(prompt(“what is your first number”));
      var num2 = Number(prompt(“what is your second number”));
      var num3 = num1 + num2;
      console.log(num3);
      console.log(num1 + num2);
      }

    addNums();

    • For loop

    • let 语句声明一个块级作用域的本地变量，并且可选的将其初始化为一个值。

    • let允许你声明一个作用域被限制在 块级中的变量、语句或者表达式。与 var 关键字不同的是， var声明的变量只能是全局或者整个函数块的。 var 和 let 的不同之处在于后者是在编译时才初始化;

    • for (let num = 1; num < 100; num++) {
      console.log(num);
      }
      这个代码执行的结果是打印1到99的输出
      如果要打印一样的输出
      for (let num = 1; num <= 100; num++) {
      console.log(num);
      }
      num的条件不一样
      因为num++是循环执行之后执行的
